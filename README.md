# Project: LED Light Cycle Wall Display

### Getting Started

- Step 1: Installation
Download or pull the code from this repo to your machine.

- Step 2: Assemble the circuit
Assemble the circuit following the diagram.png found in the directory.

- Step 3: Load the code
Upload the code contained in this sketch on to your board using the ARDUINO IDE.exe or browser program.
